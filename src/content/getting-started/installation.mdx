# Installation

## Requirements

- [Node.js](https://nodejs.org/en/) (v14.15.4 or higher)
- [npm](https://www.npmjs.com/) (v6.14.10 or higher)
- [git](https://git-scm.com/)
- Knowledge with [React](https://reactjs.org/)
- Knowledge with [Next.js](https://nextjs.org/)
- Knowledge with [MDX](https://mdxjs.com/) & [Markdown](https://www.markdownguide.org/)

To create your own documentation using Naest you need to clone the repository and install the dependencies.

```bash
# clone from github
git clone https://github.com/bdbch/naest.git

# remove the .git folder and reinitialize it
rm -rf .git
git init

# install dependencies
npm install
```
