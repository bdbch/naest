# Changelog

This is the changelog for the Naest template. Here you can find all the changes that have been made to the template.

## [1.0.0-alpha.4](https://github.com/bdbch/naest/releases/tag/v1.0.0-alpha.4) (2024-05-01)

- Add support for Sitemap generation
- Add support for Robots generation
- Added environment variables

## [1.0.0-alpha.3](https://github.com/bdbch/naest/releases/tag/v1.0.0-alpha.3) (2024-04-27)

- Added code highlighting via Highlight.js (hljs)
- Added code block copy functionality

## [1.0.0-alpha.2](https://github.com/bdbch/naest/releases/tag/v1.0.0-alpha.2) (2024-04-27)

- Added GFM support for MDX (Github Flavored Markdown) including tables, task lists, and strikethrough

## [1.0.0-alpha.1](https://github.com/bdbch/naest/releases/tag/v1.0.0-alpha.1) (2024-04-27)

- Initial alpha release of the Naest template
- Includes basic documentation setup with Next and MDX
- Automatic and recursive sidebar fetching
- Table of Content implementation
- Dark mode
